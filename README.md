# React + Vite
# Lista de Tareas React

## Descripción

Esta aplicación permite agregar y eliminar tareas de una lista utilizando React.

## Tecnologías utilizadas

* React
* Vite
* CSS

## Componentes

### App.jsx

Componente principal encargado de:

* Gestionar el estado de las tareas.
* Agregar nuevas tareas.
* Eliminar tareas existentes.

### TaskList.jsx

Componente encargado de mostrar la lista de tareas y el botón para eliminarlas.

## Estado

Se utiliza useState para:

* Almacenar el texto ingresado.
* Mantener la lista de tareas.

## Instalación

```bash
npm install
```

## Ejecución

```bash
npm run dev
```

## Funcionalidades

* Agregar tareas.
* Eliminar tareas.
* Interfaz simple y responsiva.

## Decisiones de diseño

Se utilizó una interfaz minimalista para facilitar la experiencia del usuario y mantener el enfoque en la funcionalidad principal de la aplicación.
